# **App Name**: Lexica

## Core Features:

- Hybrid Legal Retrieval: Combines BM25 and vector search for precise statute and case retrieval, powered by LLM tool usage. Integrates legal knowledge graph for authority weighting.
- AI-Powered Summarization: Provides concise summaries of legal documents and cases using LLM.
- Drafting Assistant: Assists in drafting petitions and legal documents using LLM, incorporating retrieved citations and compliance checks.
- Interactive Timeline Generation: Generates case timelines with key events and expected next steps, informed by a legal knowledge graph.
- Document Analysis Tool: Annotates uploaded documents and suggests redline edits with relevant precedent using LLM tool to locate supporting precedent.
- Role-Based Login: UI adapts to user role (Advocate, Student, Public) for tailored information retrieval and access.
- RAG-Powered Search Badge: Visual cue indicating search results are grounded with source citations.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey authority and expertise.
- Background color: Light Gray (#E6E6E6) for a clean and neutral backdrop.
- Accent color: Gold (#FFD700) for highlighting key information and calls to action.
- Body font: 'PT Sans', a humanist sans-serif that is modern with a little warmth.
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel.
- Use clear and professional icons for document types, actions, and search filters.
- Balanced layout with a focus on readability; incorporates a centered chat window and sidebar for quick access.
